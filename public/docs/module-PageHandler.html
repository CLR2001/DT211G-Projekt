Module: PageHandler

This module manages the Single Page Application (SPA) logic.
Key features:
1. Automatically imports page modules using Vite's glob import.
2. Manages dynamic content rendering without page reloads.
3. Synchronizes with the Browser History API (pushState/popstate).
