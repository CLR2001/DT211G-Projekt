Module: ImageOptimizer

This module automates the image processing pipeline to ensure high performance and retina support.
Key features:
1. Converts images to modern formats (AVIF, JPG, WebP).
2. Optimizes and compresses all image files.
3. Generates responsive srcsets for various screen widths.
